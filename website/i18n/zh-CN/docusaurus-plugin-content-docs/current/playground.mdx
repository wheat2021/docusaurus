# Playground

Playground 允许您**在浏览器中运行 Docusaurus，无需安装任何东西**！

它们主要用于：

- 测试 Docusaurus
- 报告错误

使用 [docusaurus.new](https://docusaurus.new) 作为易于记忆的快捷方式。

请从下面可用的选项中选择一个。

```mdx-code-block
import {PlaygroundCardsRow} from '@site/src/components/Playground';

<PlaygroundCardsRow />
```

:::tip

为了方便起见，我们会在您下次访问 [docusaurus.new](https://docusaurus.new) 时记住您的选择。

:::
