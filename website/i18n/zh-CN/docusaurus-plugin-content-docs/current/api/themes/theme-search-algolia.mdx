# 📦 theme-search-algolia

该主题提供了一个可轻松与 Algolia DocSearch 集成的 `@theme/SearchBar` 组件。与 `@docusaurus/theme-classic` 结合使用，它提供了一个非常简单的搜索集成。你可以在[搜索](../../search.mdx)文档中阅读更多信息。

```bash npm2yarn
npm install --save @docusaurus/theme-search-algolia
```

该主题还在 `/search` 路径添加了支持 OpenSearch 的搜索页面（作为可 swizzle 的 `SearchPage` 组件）。你可以通过 `themeConfig.algolia.searchPagePath` 更改此默认路径。使用 `false` 可禁用搜索页面。

:::tip

如果你已经安装了 `@docusaurus/preset-classic`，则无需将其作为依赖项安装。

:::
