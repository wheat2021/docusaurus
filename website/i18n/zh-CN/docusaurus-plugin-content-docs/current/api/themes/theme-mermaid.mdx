# 📦 theme-mermaid

该主题提供了一个由 [mermaid](https://mermaid-js.github.io/) 驱动的 `@theme/Mermaid` 组件。你可以在[图表](../../guides/markdown-features/markdown-features-diagrams.mdx)文档中阅读更多信息。

```bash npm2yarn
npm install --save @docusaurus/theme-mermaid
```

## 配置 {#configuration}

```js title="docusaurus.config.js"
export default {
  themes: ['@docusaurus/theme-mermaid'],
  // 为了让 Markdown 中的 Mermaid 代码块正常工作，
  // 你还需要通过此选项启用 Remark 插件
  markdown: {
    mermaid: true,
  },
};
```
