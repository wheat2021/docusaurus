# 📦 theme-live-codeblock

该主题提供了一个由 [react-live](https://commerce.nearform.com/open-source/react-live/) 驱动的 `@theme/CodeBlock` 组件。你可以在[交互式代码编辑器](../../guides/markdown-features/markdown-features-code-blocks.mdx#interactive-code-editor)文档中阅读更多信息。

```bash npm2yarn
npm install --save @docusaurus/theme-live-codeblock
```

### 配置 {#configuration}

```js title="docusaurus.config.js"
export default {
  plugins: ['@docusaurus/theme-live-codeblock'],
  themeConfig: {
    liveCodeBlock: {
      /**
       * 实时演示区域的位置，在编辑器上方或下方
       * 可选值: "top" | "bottom"
       */
      playgroundPosition: 'bottom',
    },
  },
};
```
