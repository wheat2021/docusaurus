# Docusaurus 主题

我们提供官方的 Docusaurus 主题。

## 主要主题 {#main-themes}

主要主题为 [文档](../plugins/plugin-content-docs.mdx)、[博客](../plugins/plugin-content-blog.mdx) 和 [页面](../plugins/plugin-content-pages.mdx) 插件实现用户界面。

- [@docusaurus/theme-classic](./theme-classic.mdx)
- 🚧 其他主题正在规划中

:::warning

我们的目标是让所有主题共享完全相同的功能、用户体验和配置。

应该只有 UI 设计和底层的样式框架会有所不同，并且你应该能够轻松地更换主题。

我们还没实现这个目标：目前只有 classic 主题是生产可用的。

:::
