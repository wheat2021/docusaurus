# 📦 plugin-google-tag-manager

import APITable from '@site/src/components/APITable';

一个用于将 [Google 标签管理器 (gtm.js)](https://developers.google.com/tag-platform/tag-manager) 添加到 Docusaurus 网站的插件。将此插件与标准的 [gtag 插件](./plugin-google-gtag.mdx) 结合使用，可以深入分析用户如何使用你的网站。

:::tip

你可以使用 [Google 的 Tag Assistant](https://tagassistant.google.com/) 工具来检查标签管理器是否设置正确！

:::

:::warning 仅限生产环境

该插件在开发环境中始终处于非活动状态，**仅在生产环境中激活**，以避免污染分析统计数据。

:::

## 安装 {#installation}

```bash npm2yarn
npm install --save @docusaurus/plugin-google-tag-manager
```

:::tip

如果你使用预设 `@docusaurus/preset-classic`，则无需将此插件作为依赖项安装。

你可以通过[预设选项](../../using-plugins.mdx#docusauruspreset-classic)配置此插件。

:::

### 配置示例 {#ex-config}

你可以通过预设选项或插件选项来配置此插件。

:::tip

大多数 Docusaurus 用户通过预设选项配置此插件。

:::

```js config-tabs
// 预设选项: googleTagManager
// 插件选项: @docusaurus/plugin-google-tag-manager

const config = {
  containerId: 'GTM-12345',
};
```
