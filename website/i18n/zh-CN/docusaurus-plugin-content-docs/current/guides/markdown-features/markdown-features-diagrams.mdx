# 图表

可以在代码块中使用 [Mermaid](https://mermaid-js.github.io/mermaid/) 来渲染图表。

## 安装 {#installation}

```bash npm2yarn
npm install --save @docusaurus/theme-mermaid
```

通过在 `docusaurus.config.js` 中添加插件 `@docusaurus/theme-mermaid` 并将 `markdown.mermaid` 设置为 `true` 来启用 Mermaid 功能。

```js title="docusaurus.config.js"
export default {
  markdown: {
    mermaid: true,
  },
  themes: ['@docusaurus/theme-mermaid'],
};
```

## 用法 {#usage}

添加一个语言为 `mermaid` 的代码块：

````md title="Mermaid 图表示例"
```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```
````

```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```

有关 Mermaid 语法的更多信息，请参阅 [Mermaid 语法文档](https://mermaid-js.github.io/mermaid/#/./n00b-syntaxReference)。

## 主题 {#theming}

可以通过在 `docusaurus.config.js` 的 `themeConfig` 中设置 `mermaid.theme` 的值来更改图表的深色和浅色主题。您可以为浅色和深色模式设置不同的主题。

```js title="docusaurus.config.js"
export default {
  themeConfig: {
    mermaid: {
      theme: {light: 'neutral', dark: 'forest'},
    },
  },
};
```

有关 Mermaid 图表主题的更多信息，请参阅 [Mermaid 主题文档](https://mermaid-js.github.io/mermaid/#/theming)。

## Mermaid 配置 {#configuration}

`mermaid.options` 中的选项将直接传递给 `mermaid.initialize`：

```js title="docusaurus.config.js"
export default {
  themeConfig: {
    mermaid: {
      options: {
        maxTextSize: 50,
      },
    },
  },
};
```

有关可用的配置选项，请参阅 [Mermaid 配置文档](https://mermaid-js.github.io/mermaid/#/./Setup?id=configuration)和 [Mermaid 配置类型](https://github.com/mermaid-js/mermaid/blob/master/packages/mermaid/src/config.type.ts)。

## 动态 Mermaid 组件 {#component}

要生成动态图表，您可以使用 `Mermaid` 组件：

```mdx title="动态 Mermaid 组件示例"
import Mermaid from '@theme/Mermaid';

<Mermaid
  value={`graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;`}
/>
```
