# 数学方程式

import BrowserWindow from '@site/src/components/BrowserWindow';

可以使用 [KaTeX](https://katex.org) 渲染数学方程式。

请参阅[下文](#configuration)了解如何启用它们。

## 用法 {#usage}

更多详情，请阅读 [KaTeX](https://katex.org) 文档。

### 行内 {#inline}

通过在 `$` 之间包裹 LaTeX 方程式来编写行内数学方程式：

```latex
Let $f\colon[a,b]\to\R$ be Riemann integrable. Let $F\colon[a,b]\to\R$ be
$F(x)=\int_{a}^{x} f(t)\,dt$. Then $F$ is continuous, and at all $x$ such that
$f$ is continuous at $x$, $F$ is differentiable at $x$ with $F'(x)=f(x)$.
```

### 块 {#blocks}

对于方程式块或显示模式，请使用换行符和 `$$`：

```latex
$$
I = \int_0^{2\pi} \sin(x)\,dx
$$
```

## 启用数学方程式 {#configuration}

启用 KaTeX：

1.  安装 `remark-math` 和 `rehype-katex` 插件：

    ```bash npm2yarn
    npm install --save remark-math@6 rehype-katex@7
    ```

    :::warning

    请确保为 Docusaurus v3（使用 MDX v3）使用 `remark-math 6` 和 `rehype-katex 7`。我们不保证其他版本能够正常工作。

    :::

2.  这两个插件**仅作为 ES 模块提供**。我们建议使用 **ES 模块** 配置文件：

    ```js title="ES 模块 docusaurus.config.js"
    // highlight-start
    import remarkMath from 'remark-math';
    import rehypeKatex from 'rehype-katex';
    // highlight-end

    // highlight-start
    export default {
      presets: [
        [
          '@docusaurus/preset-classic',
          {
            docs: {
              path: 'docs',
              // highlight-start
              remarkPlugins: [remarkMath],
              rehypePlugins: [rehypeKatex],
              // highlight-end
            },
          },
        ],
      ],
    };
    ```

## 自托管 KaTeX 资源 {#self-hosting-katex-assets}

对于流行的库和资源，从 CDN 源加载样式表、字体和 JavaScript 库是一个好习惯，因为这可以减少您需要托管的资源数量。如果您希望自托管 `katex.min.css`（以及所需的 KaTeX 字体），您可以从 [KaTeX GitHub releases](https://github.com/KaTeX/KaTeX/releases) 下载最新版本，解压并复制 `katex.min.css` 和 `fonts` 目录（仅 `.woff2` 字体类型就足够了）到您站点的 `static` 目录中，然后在 `docusaurus.config.js` 中，将样式表的 `href` 从 CDN URL 替换为您的本地路径（例如 `/katex/katex.min.css`）。

```js title="docusaurus.config.js"
export default {
  stylesheets: [
    {
      href: '/katex/katex.min.css',
      type: 'text/css',
    },
  ],
};
```
